# JMEV Tunisia - Site Web Officiel

Site web moderne et responsive pour la commercialisation des véhicules électriques JMEV en Tunisie.

## Technologies

- Next.js 14 (App Router)
- React 18
- TypeScript
- Tailwind CSS

## Installation

```bash
npm install
```

## Développement

```bash
npm run dev
```

Ouvrez [http://localhost:3000](http://localhost:3000) dans votre navigateur.

## Structure du Projet

- `/app` - Pages et routes Next.js
- `/components` - Composants réutilisables
- `/lib` - Utilitaires et helpers
- `/public` - Assets statiques

## Pages Principales

- `/` - Page d'accueil
- `/models` - Liste des modèles
- `/models/[slug]` - Détails d'un modèle
- `/offers` - Offres et financement
- `/charging` - Chargement et autonomie
- `/about` - À propos
- `/news` - Actualités
- `/contact` - Contact
